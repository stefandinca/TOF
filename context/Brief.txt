Twist of Fate
Board Game Library & Inventory System
Aka Board Game Menu
Design & Implementation Brief

1. Project Overview
Twist of Fate requires a Board Game Library system that serves two distinct audiences:
Customers (Public-Facing UI)
Browse the in-store board game library to discover games available to play or purchase.
Staff (Internal UI)
Manage inventory quantities, add and maintain games, and track inventory changes with accountability.
This system will integrate into the Twist of Fate website and should feel like a modern board game caf√© discovery experience, not a traditional inventory database.
The desired baseline experience is similar to:
https://app.questcentral.ca/games
with specific enhancements outlined below.

2. Core Goals
Fast, intuitive browsing for customers
Simple, low-friction inventory management for staff
Clear distinction between Library and Retail availability
Manual control over metadata (auto-scraping override if possible)
Architected to support future POS / Shopify integration

3. Scope & Phasing

4. Data Source & Structure
The system will be built from a Board Game Library spreadsheet, which serves as:
Initial data import to Firebase
Ongoing admin reference (not real-time once live)
Ability to Export live library to spreadsheet for historical tracking

9. Final Instruction to Designer

fewer clicks, instant feedback, and visible state changes.
If a feature is ambiguous, clarity and usability take precedence over complexity.
